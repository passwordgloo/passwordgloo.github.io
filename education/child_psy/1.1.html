    # 一、研究对象了解

    儿童心理学或儿童发展心理学是一门研究儿童的心理随着时间的流逝（年龄的增长）在心理上和行为上发生的变化的学科。

    儿童。个体从出生到成熟这段时间内，在心理上和行为上发生的变化是儿童心理学研究的对象。这段时间里的个体就是儿童（广义的）。

    成熟。儿童期的确定与成熟有关，何谓“成熟”？一般指个人心身发育过程的完成。心身成熟意味着儿童期的结束，成年期的开始，或继续向成年期过渡。个体生理上的成熟时间很明确-性发育的完成。可是心理上的确切的成熟时间很难确定。因为心理活动本身是多侧面、多过程的。再说，生理和心理的发展虽然互相关联，但各系统成熟的时间并不同步，即使是身体各系统，其发展也是多元化的，因此对成熟年龄的估计也不易一致，不过，从总体上来说，国内外一般都认同的个体成熟的年龄在17、18岁左右。但随着社会的发展，儿童生理成熟年龄的前倾，心理成熟年龄的延缓或推迟，欧美一些国家已把儿童发展心理学研究的对象年龄范围延伸到19、20、22岁。这也反映现在的儿童成为成人需要经过更长的过渡期或准备期。

    儿童发展心理学与发展心理学。心理学是研究人的行为和思维的科学。发展心理学是心理学的一个分支，也研究人的行为和思维。但两者的具体研究对象并不相同。前者主要研究各种心理活动过程或行为本身的性质或特征，如就记忆来说，先要回答什么是记忆，它是一种怎样的心理过程，记忆有哪几种，记忆的运作历程如何进行，怎样测量记忆，影响记忆的因素有哪些；而发展心理学则是研究各种心理活动和行为伴随着个体年龄（时间）的增长（流逝）在什么时候发生了什么变化，如何变化（顺序、规律），为什么会产生这些变化，影响这些变化的因素又是什么（原因、机制），如何促进心理和行为的发展（指导）。“一生发展心理学”或“发展心理学”是研究个体生命全程的心理发生、发展的过程与规律的学科，“儿童心理学”或“儿童发展心理学”只研究人的生命的早期阶段-儿童心理发生、发展的过程与规律，以及各年龄阶段儿童年龄特征的学科。很明显，儿童发展心理学是发展心理学的一个分支。

    发展。发展心理学和发展心理学的各个分支都研究个体心理和行为的发展。何谓发展？发展指从受精卵形成到死亡，个体身上所发生的系统的、连续的变化过程。发展就意味着变化。但不是所有的变化都是发展。

    发展的变化有三个特点：①系统性。即是有序的、有规律可循的、模式化的，因而也是可预测的；②连续性。即变化中依然保持着原有的成分或样子，如从一个襁褓中的孩子到成为咿呀学语的孩子，到蹒跚学步的孩子，到后来进入幼儿园、小学、初中、高中。变了吗？肯定变了。从外形到语言，到思维方式、情绪、人格都发生了变化。可是他（她），还是他（她），不是别人（她）；③持久性。在时间上变化有一个相对较长的持续时间，而不是暂时性的（如疾病引发的变化），或瞬息即变的（如由突发事件引起的变化）。

    1987年保罗·巴尔特斯（Paul Baltes）提出了生命全程发展的七个关键性的假设（Baltes等，1998），对人的发展作了更全面的诠释：

    1. 发展是一个持续终生的过程。
    2. 发展是多面的。在人类生命的过程中的任何一个年龄阶段都会有一些能力在增加，一些能力在消退，而有一些能保持经久不衰。
    3. 发展同时涉及丧失和获得或消退和增长。
    4. 发展的可塑性是终生的。即使是老年人，通过训练和练习，也可能重新获得一些已经失去的能力。
    5. 发展受到个体所处的历史和文化背景的影响。
    6. 发展受到多种因素的影响，既有内部因素，又有外部因素，也包括生物因素和环境因素。发展是处在变化中的个体和正在变化着的环境交互作用的产物。
    7. 要理解发展必须结合多门学科。因为人类发展受到从生物化学反应到历史事件的各方面因素的影响，在一个学科中是不可能找到所有答案的。只有把相关学科的观点和研究工具结合才能对人类的发展有一个完整的理解。

    把个体的发展划分为生理的、认知的、人格和社会性发展三个过程，这只是为了描述和研究上的需要，事实上，三个过程本身都是互相纠缠在一起、密不可分的。就以婴儿用微笑回应母亲的爱抚动作来说，这么简单的一个回应动作就包含了生理发展过程（身体对抚摸的感知和回应能力）、认知发展过程（理解母亲这一行为的意图）和社会情绪发展过程（微笑通常表示积极的情绪，并能帮助婴儿与他人建立依恋关系）。 由此可见，发展的每一步都包含着生理过程、认知过程、人格和社会性发展过程三者复杂的相互作用

    发展阶段和年龄标准 。每个人的发展旅程与其他人都有着相似和不同之处。一些发展心理学家对发展历程的独特性感兴趣，另一些发展心理学家则对发展的共同性或普遍性很关注。不论是平凡的抑或是超凡的人都会经历着一条相似的生命发展轨迹。这条发展轨迹或生命历程包含着生理的发展过程、认知的发展过程、社会情绪和人格的发展过程，以及其他方面的变化

    # 二、研究内容简答

    学科研究的内容是由研究的目标所决定。儿童发展心理学研究的目标是：描述儿童的发展、解释儿童的发展和优化儿童的发展，而描述和解释儿童的发展的最终目的就是为了优化儿童的发展,让每一个儿童能健康地发展，达到最佳的发展。

    # 描述儿童发展的普遍模式

    描述是所有科学的开端。儿童心理学首先要研究的内容就是要描述一个个软弱无助的新生儿是怎样一步一步地成长、变化发展起来的。他们的身体和动作是怎样发展变化的？他们的语言是怎样形成、发展的？他们的感知觉、记忆、思维、创造性又是怎样变化发展的，他们什么时候开始有了情绪？婴幼儿的情绪、小学生的情绪、青少年的情绪有什么联系，有什么区别？发生了哪些变化？儿童什么时候开始有了自我意识？不同年龄儿童的自我意识有什么特点？什么时候出现了人与人之间的社会互动？婴幼儿的社会互动是怎样进行的？小学生、中学生又是怎样实现社会互动的呢？童年期儿童、少年期儿童和青年早期的儿童对同伴、对同性和异性同伴的交往有了什么变化？儿童的道德是怎样发展变化的？不同年龄的儿童是如何判断行为的是非好恶的？儿童什么时候开始有了羞愧感、同情感、美感及荣誉感？儿童有“反抗期”吗？什么年龄？什么特点？诸如此类的问题都是为了绘制出一条真实反映儿童心理发展轨迹的模式。真正的心理发展模式应该具有普遍意义，即能反映生活在不同社会文化背景下的所有儿童共同具有的发展过程。

    # 揭示儿童心理发展的原因

    儿童发展心理学的第二个研究内容就是要对研究者所描述的儿童发展模式加以验证和解释，解释模式背后儿童心理发展的原因。例如，儿童是怎样获得语言的？为什么儿童在短短的几年内就能基本上掌握本民族的语言？为什么不同民族或不同国家的儿童，尽管他们的母语都不相同，可是语言发展都会经历相似的阶段呢？这里有没有共同的先天的机制呢？如果说皮亚杰所揭示的儿童思维发展阶段就是思维发展的普遍模式，那么为什么必须要经过这几个阶段呢？思维阶段不变地推移的原因是什么呢？对上述内容的研究和回答标志着儿童发展心理科学的成熟水平。许多发展学家根据收集到的资料提出自己的假设，然后通过进一步的观察、实验等的验证，构建各具特色的儿童心理发展的理论。

    # 解释和测量个别差异

    尽管每个儿童心理发展经历的阶段、或发展变化的模式基本上是相同的，但每个儿童心理发展的速度、心理发展最后达到的水平、各种心理过程和行为特点并不相同。有的儿童外向、热情、喜爱交际，有的儿童则是内向、冷漠、讨厌与人交往，有的儿童擅长音乐绘画，有的儿童喜欢数学物理，善于抽象的思考，有的儿童攻击性很强，有的儿童乐于助人，有的软弱自卑，有的孤傲自负······儿童与儿童间存在这样那样的差异是怎样造成的？如何测量这些差异呢？这也是儿童发展心理学研究的基本内容之一。回答这些问题有助于我们更好地解释儿童心理发展的原因以及心理发展所需要的条件。有一部分发展学家更关注发展的普遍模式研究，也有一部分发展学家从事发展的个别差异的研究。这样的研究应该是相得益彰。但传统上个别差异有一种“缺陷式”的理解，往往把个别差异看成是某种缺陷或异常的指标。当今发展系统论则把个别差异看成是“发展的多样性”，是人类生命历程特有的表现，是人类发展的重要财富。“因为它界定了人类生命最优化之潜在物质基础的变异范围，它使人们利用它以实现自身积极、健康的毕生发展成为可能。”

    # 探究环境对发展的影响

    每个儿童一来到这个世界（实际上是从受精卵形成的一瞬间）就开始受到了他所生活的环境里多种多样因素的影响。而且随着年龄的增长、活动能力的提高，影响儿童行为的环境因素会越来越多。这类环境有时候被描述为儿童行为的生态圈。不同的生态环境对儿童的发展会产生什么样的影响呢？这又是一个重要的研究课题。在这些生态环境中，与儿童接触时间最长、影响最大的可能就是家庭、学校和社区了。如不同的家庭结构（完整家庭或不完整家庭、主干家庭或核心家庭、独生子女家庭或多子女家庭，还有重组家庭等）对儿童的发展会有什么不同的影响？同样结构的家庭，父母的教养方式不同（如专制型、民主型、溺爱型、冷漠型）对儿童个性的形成会有什么影响？家庭的物理环境（如高层住宅、多层住宅、平房、独立别墅、家庭装潢、生活设施、玩具、电视、手机、电脑、iPhone和互联网等电子产品等等）对儿童有什么影响？父母的文化水平、父母的个性、生活习惯（如运动、抽烟、酗酒、网瘾、赌博、吸毒等）、父母的职业、家庭的经济水平会对儿童有影响吗？儿童进了学校以后，教师的人格特质、对儿童的态度、教师的教学教育水平、学校的校风、师生关系、同伴关系怎样影响儿童的成长？让儿童进寄宿制学校好，还是不寄宿好，或者就在家庭接受教育好？婴幼儿全托好，还是半托或日托好，或在家抚养好？社区环境中文化娱乐设施、健身场地、绿化水平、噪音度、文明风气、邻里关系、社区社会支持体系等因子怎样影响身在其中的成人和他们的孩子？此外，还有一个无处不在、无时不有的社会大环境--社会文化历史背景对儿童的影响。

    从社会大环境来看，电视里天天播放吃、喝、玩、乐、时尚的诱惑人的广告、青少年崇拜的明星的奢华与绯闻、网络上传播的信息（包括各种儿童不宜的色情、拐骗，不顾新闻道德的某种报道）等等会对成长中的儿童造成什么伤害吗？在网络化的时代，网络对不同年龄的儿童会产生什么影响？为什么有些儿童成了“网瘾”，成了现代化工具的牺牲品？

    了解儿童生态环境对儿童发展的影响可以进一步揭示儿童发展的原因和机制，也能为科学地指导儿童健康地发展创造条件。

    # 提出帮助与指导儿童发展的具体方法

    描述儿童心理发展的原因和机制，测量和解释发展的个别差异，探讨系统生态环境对发展的影响，目的只有一个，就是为了帮助儿童顺利地度过每个发展阶段，帮助儿童解决在发展中遇到的困难或暂时的障碍。我们不仅要解决发展是什么、为什么的问题，还要解决怎么指导儿童、怎样使儿童的心理发展达到最优化。理论的建构不仅仅是为了解释种种心理过程和行为发生发展的原因，还应该结合社会实际来指导儿童健康地发展。前者的研究可称为基础理伦研究，后者的研究可以称为应用性研究。这两方面的研究都是促进和繁荣儿童心理科学不可或缺的。而且这两部分的研究不是绝对分离的。例如，有些心理学家试图利用婴儿对不同声音呈现时的吸吮模式的变化来测定还不会说话的婴儿是否具有区别不同语音的能力。几年后，临床心理学家用这种技术来确定儿童的听觉是否有损伤，而这可能并非是前面从事这项研究的心理学家的初衷。

    自1996年Stokes提出了“应用的基础研究”概念之后，基础研究与应用研究已不再是泾渭分明的两个方面，而是成了“沟通基础研究与活生生的任何活动之间的管道”。儿童发展心理学家也不再只关心认知与情意功能割裂的某些方面，而是去拓展“教育或临床干预以及课本、软件、课程及媒体如何设计等实践的专业性功能”，这表明现在的儿童发展心理学更加重视满足社会需要的应用的基础研究。儿童发展心理学理论的研究或应用都需要发展理论的指导，而发展理论本身也需要、也只能通过实践的应用才能得以检验修正、丰富和发展。
