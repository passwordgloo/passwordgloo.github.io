                  # 包含GUI程序

                  Ubuntu引入gnome桌面满足颜值党的喜好，同时也增加了系统负担，本着返璞归真的态度，保留图形界面，可以设置无桌面启动,可以参考无桌面环境启动GUI程序open in new window

                  # 不含GUI程序

                  对图形窗口程序无需求，习惯命令行终端操作可以卸载桌面

                  该操作无法用命令行方式启动图形窗口程序

                  如果后悔的话，执行sudo apt install gnome-shell

                  sudo apt-get remove gnome-shell			//卸载gnome-shell
sudo apt-get remove gnome --purge 		//如果有gnome的话，卸载gnome及其配置
    
----------------------------清除残余------------------------------
sudo apt-get autoremove 				//清除依赖项
sudo apt-get autoclean					//清除缓存
sudo apt-get update						//可选操作
